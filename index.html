About Vaultic Crypto Engine

Vaultic Crypto Engine is a high-performance cryptographic library that provides secure RSA operations with support for both native Rust and WebAssembly environments.

Enhanced Security

Includes mitigations against timing attacks and implements best-practice RSA padding

High Performance

Built with Rust and compiled to WebAssembly for optimal speed and security

Cross-Platform

Works seamlessly in browsers, Node.js, and native Rust environments

Developer Friendly

Simple API with comprehensive documentation and examples

Handles Data of Any Size

Automatically switches between direct RSA and hybrid RSA+AES encryption based on data size

Key Features:

  • RSA Key Generation: Secure 2048-bit RSA key pairs
  • Encryption/Decryption: RSA-PKCS#1 with proper padding
  • Base64 Encoding: For easy transport and storage
  • Attack Mitigations: Includes protections against the Marvin timing attack

Learn more about Vaultic Crypto Engine on GitHub.

Adaptive Encryption Technology

Vaultic's intelligent encryption engine automatically selects the optimal encryption method based on your data:

Direct RSA

For small messages (< 245 bytes), Vaultic uses direct RSA-2048 encryption for simplicity and security.

Hybrid RSA+AES

For larger data, Vaultic seamlessly switches to hybrid encryption using AES-256-GCM with an RSA-encrypted key.
